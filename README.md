

#Installation 
In order to start this project for the first time, you should run

```
npm install

```
To start this project, run
```
npm start
```
To start with nodemon, use 
```
npm run dev 
```
## Environment variables
This project uses environment variables as follows 
```
MYSQL_DATABASE=""
MYSQL_USER=""
MYSQL_PASSWORD=""
MYSQL_HOST=""
PORT=""
```
